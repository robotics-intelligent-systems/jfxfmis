# Farm Management Information System

LiteFarm is a farm management tool built for current and aspiring sustainable farmers. farmOS is a web-based application for farm management, planning, and record keeping. AgML is a centralized framework for agricultural machine learning. OpenHydroponics is an project aimed at automating and optimizing hydroponic systems. FarmBot's web based user interface, a RESTful JSON API, and a Dockerized MQTT server. Tania is a management software. Ekylibre is a farm management information system (aka FMIS) web application. GeoServer is an open-source server for sharing geospatial data. DISPLACE is a dynamic, individual-based model for spatial fishing planning and effort displacement integrating underlying fish population models. PeskAAS: A near-real-time, monitoring and analytics system for small-scale fisheries. The Open Research Vessel Data Acquisition System (OpenRVDAS) is a software framework used for building custom data acquisition systems (DAS). Slick is an R package designed for visualizing and exploring results of a Management Strategy Evaluation (MSE). The FSA package provides R functions to conduct typical introductory fisheries analyses. open-source prototypes for fish farming applications based on LoRaWAN communications. A Modelica library for simulation of recirculating aquaculture systems. Farm Data Relay System is an easy way to communicate with remote IoT devices. Odoo's Manufacturing Resource Planning (MRP) module is a comprehensive solution designed to streamline manufacturing processes. The GIS Tools for Hadoop are a collection of GIS tools that leverage the Spatial Framework for Hadoop for spatial analysis of big data. The Apache Hive (TM) data warehouse software facilitates reading, writing, and managing large datasets residing in distributed storage using SQL.
