# Farm Management Information System

DMsan is an open-source platform for decision-making for sanitation and resource recovery systems. FarmBot's web based user interface, a RESTful JSON API, and a Dockerized MQTT server. Tania is a free and open source farm management software. Ekylibre is a farm management information system (aka FMIS) web application. GeoServer is an open-source server for sharing geospatial data. Fleetbase is a modular logistics and supply chain operating system. Optaplanner is a fast AI constraint solver for software developers. The Open Research Vessel Data Acquisition System (OpenRVDAS) is a software framework used for building custom data acquisition systems (DAS). FishCam: A low-cost open source autonomous camera for aquatic research. Fish Detection (Segmentation) & Classification models and training scripts. OpenFish is an open-source system written in Golang for classifying marine species. R package for accessing data from Global Fishing Watch APIs. Training data detecting fishing behaviour in vessel movement patterns. An open-source platform to predict and visualize shark foraging hotspots using a novel habitat suitability model (TCHI) powered by NASA satellite data. open-source prototypes for fish farming applications based on LoRaWAN communications. A Modelica library for simulation of recirculating aquaculture systems. Odoo's Manufacturing Resource Planning (MRP) module is a comprehensive solution designed to streamline manufacturing processes. The GIS Tools for Hadoop are a collection of GIS tools that leverage the Spatial Framework for Hadoop for spatial analysis of big data. The tools make use of the Geoprocessing Tools for Hadoop toolbox, to provide access to the Hadoop system from the ArcGIS Geoprocessing environment. The Apache Hadoop project develops open-source software for reliable, scalable, distributed computing. The Apache Hive (TM) data warehouse software facilitates reading, writing, and managing large datasets residing in distributed storage using SQL.
